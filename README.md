## EN

**3DPC (3D Print Calculator) is a desktop application for accurate 3D printing cost calculation.**

It is designed for makers, studios, and small manufacturing setups that need a fast and transparent way to calculate production costs and final pricing.

**Features**

‚úî Cost calculation based on weight, print time, and quantity

‚úî Material support (PLA / PETG / ABS)

‚úî Electricity cost calculation based on printer power

‚úî Hourly equipment depreciation

‚úî Model complexity factor

‚úî Additional services: modeling, post-processing, other works

‚úî Prototype (test print) support

‚úî Detailed cost breakdown by category

‚úî Multiple printers with individual settings

‚úî Dark and light themes

‚úî Russian and English interface

‚úî Flexible settings

‚úî Filament price per material type

‚úî Printer power and electricity tariff

‚úî Depreciation in $ / hour

‚úî Final price multiplier

‚úî Separate profiles for different printers
